# Project Name
Cinema Room Manager

## Description
A program designed to manage and organize cinema room layouts, including seating arrangements and ticket reservations.

## Author
Roy Aimo

## Requirements
- Java 17 or higher
- Gradle (you can use the included Gradle wrapper)

## Project Structure
- `Cinema Room Manager/` → Source code (`src`)
- `build.gradle` → Gradle build configuration
- `gradlew`, `gradlew.bat` → Gradle wrapper scripts
- `settings.gradle` → Project settings
- `build/` → Compiled files and generated `.jar`
- `.idea/` → IntelliJ IDEA project settings (optional)

> Note: The `.jar` file is not included. You can build it locally using Gradle.
